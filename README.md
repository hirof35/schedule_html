Ultimate Schedule App
FullCalendar 6を使用した、シンプルで高機能なブラウザ完結型スケジュール管理アプリです。
サーバー不要で、ブラウザのローカルストレージを利用してデータを保存します。

✨ 主な機能
カレンダー操作: 月表示、週表示、リスト表示の切り替えが可能。

ドラッグ＆ドロップ: 予定の移動や期間の変更が直感的に行えます。

カテゴリー管理: 「仕事」「プライベート」「その他」で色分け管理。

検索機能: タイトルやカテゴリー名で予定をリアルタイムに絞り込み。

ダークモード: 目に優しいダークテーマへの切り替えに対応。

CSVエクスポート: 登録した予定をCSV形式でダウンロード可能。

デスクトップ通知: 予定の時間になるとブラウザ通知でお知らせします。

レスポンシブ対応: スマートフォンでは自動的にリスト表示に最適化されます。

🚀 使い方
index.html（このコード）をダウンロードします。

お好きなブラウザでファイルを開くだけで使用可能です。

注: 通知機能を使用する場合は、ブラウザの通知許可をオンにしてください。

🛠 技術スタック
Frontend: HTML5, CSS3 (CSS Variables), JavaScript (ES6+)

Library: FullCalendar 6.1.10

Storage: Browser LocalStorage

📂 ファイル構成
Plaintext
.
└── index.html  # 全てのロジック、スタイル、UIが含まれます
📝 注意事項
データはブラウザのローカルストレージに保存されます。ブラウザのキャッシュをクリアするとデータが消去されるため、重要なデータは定期的に「CSV保存」ボタンからバックアップを取ることを推奨します。

通知機能はブラウザを開いている間のみ動作します。
